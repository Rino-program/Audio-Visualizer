# 开发者寄语
感谢您使用 Audio Visualizer。
此应用是专注于本地播放的 F-Droid 版本。

## 关于本应用
这是一款让您在欣赏音乐文件的同时享受精美视觉效果的工具。
本应用采用开源方式开发，代码已在 GitHub 上公开。
https://github.com/Rino-program/Audio-Visualizer

## 未来计划
- 进一步优化功能
- 发展为音乐播放器
- 响应用户反馈

## 开源协议
本应用基于 MIT 许可证开源，您可以完全自由地使用、修改和分发它。

## 开发者的小请求（可选）
如果您在自己的项目或创作中使用了这款应用，或者非常喜欢它，欢迎在 X（原 Twitter）上带上标签 #AudioVisualizerFDroid 进行分享，或者提及（@）开发者 @Rinoprogram！
了解这款应用是如何被大家使用的，将成为我们持续开发的最大动力。（当然，这完全是自愿的，您完全可以在不通知的情况下自由使用！）

## 早期发布与 Beta 版分发
